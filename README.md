# Product Crawler

This project is used to crawl product information from:

- danmurphys.com.au
- bws.com.au
- liquorland.com.au

## Installation

Install Bun if you haven't already. Follow the instructions on [Bun's website](https://bun.sh/).

To install dependencies:

```bash
bun install
```

To crawl danmurphys.com.au website, run:

```bash
bun run danmurphys.com.au
```

To crawl bws.com.au website, run:

```bash
bun run bws.com.au
```

To crawl liquorland.com.au website, run:

```bash
bun run liquorland.com.au
```
